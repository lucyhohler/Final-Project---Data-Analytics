Meet ET in person

UFO sightings and crop circles were a big topic of discussion in the early nineties. Even though this topic has mostly gone by the wayside,
actual UFO sightings never slowed down.

Our hypothesis: If you use the right supervised machine learning prediction model to train and test our data, we can get an accuracy score of at least 90%.

To accomplish this we are using our UFO sightings dataset of the last 88 years, focused on the US only (60k+ rows of data).

To analyze, visualize and find interesting trends, we used Tableau.

To predict our data we used various machine learning models.


Machine Learning


The next step is to use Machine Learning to allow us to predict ufo sightings.
Because our data provided both our input and output variables we used supervised learning models:

  • Logistic Regression
  • KNN
  • Deep Learning
  • Neural Network

Logistic Regression

A predictive modelling algorithm that is used when the Y variable is binary categorical. That is, it can take only two values like 1 or 0. The goal is to determine a mathematical equation that can be used to predict the probability of an event. Once the equation is established, it can be used to predict the Y when only the Xâ€™s are known.
Deep Learning

A machine learning technique that teaches computers to learn by example. In deep learning, a computer model learns to perform classification tasks directly from images, text, or sound. Deep learning models can achieve state-of-the-art accuracy.
KNN
K Nearest Neighbor is an efficient algorithm used for classification. Because k can vary your results, the easiest technique for choosing a k value is to loop through a range k and calculate the score. We then choose the lowest value of k where the score starts to stabilize.
Neural Network (our best prediction model - 89.35% accuracy)

A neural network is nothing more than a bunch of neurons connected together. A neuron, the basic unit of the network, takes inputs (data), does some math with them, and produces an output. You can also apply hidden layers to have the machine do some additional calculations.


Conclusion
If you want to see UFOs the most likely state will be California during the summer at night. Look up and you might see a bright light, circle, triangle or fireball.

This finding is also backed by our machine learning models which have shown us that the neural network had the best accuracy, getting us close to our hypothesis of getting a 90% accuracy score.
